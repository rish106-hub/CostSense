System Architecture
9-Agent Choreography Pipeline
POST /ingest/* └─► Agent 01 (data_connector) raw.spend └─► Agent 02 (normalization) normalized.spend └─► Agent 03 (anomaly_detection) anomaly.detected ├─► Agent 04 (root_cause) [LLM] anomaly.enriched └─► Agent 05 (prioritization) [<1ms] anomaly.scored ╲ ╱ Agent 06 (merge) anomaly.ready └─► Agent 07 (action_dispatcher) ├─► action.approval_needed └─► action.auto_execute └─► Agent 08 (executor) Agent 09 (audit_trail) ── listens ALL 8 topics
Scoring Model
Anomaly Score (AS)
AS = (FI×0.40) + (FR×0.25) + (RE×0.20) + (SR×0.15) Range: 1–10
Action Priority (APS)
APS = AS × confidence / complexity Range: 0–10
Approval Routing
APS ≥ 4.0 AND complexity ≥ 2 → approval_needed
Complexity Tiers
<₹50K → 1 (auto) ₹50K–₹2L → 2 (Slack) ₹2L–₹10L → 3 (Finance) >₹10L → 5 (Board)
